# Contributing to BRICE Sentinel

Thank you for your interest in contributing! We welcome PRs, issues, and suggestions.

## How to Contribute
- Fork the repo and create a feature branch
- Write clear, modular, and well-documented code
- Add/maintain tests for new features or bugfixes
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md)
- Submit a pull request with a clear description

## Development
- Use Docker Compose for local development
- Run tests before submitting PRs
- Document new features in the appropriate README or API docs

## Issues
- Use GitHub Issues for bugs, feature requests, and questions
- Please provide as much detail as possible
